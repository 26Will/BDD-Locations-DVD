# 📀 Analyse et Reporting - Loueur de DVD (Base Sakila)

## 📝 Contexte du Projet
Dans le cadre d'une SAE (Situation d'Apprentissage et d'Évaluation), ce projet vise à analyser les données d'un loueur de DVD des années 2005 (Base de données *Sakila*). L'objectif est de dégager des axes stratégiques pour l'ouverture d'un nouvel espace physique convivial combinant location de DVD et petite restauration.

## 🎯 Objectifs de l'analyse
- **Vision Clients :** Identifier les meilleurs clients (montant dépensé, volume de location).
- **Vision Locations :** Déterminer les jours et les tranches horaires les plus propices à la location.
- **Catégorie de Films :** Analyser les performances par genre et identifier les films "anciens" à fort potentiel mais peu loués.

## 🛠️ Outils & Technologies Utilisés
- **SQL :** Extraction et préparation des données depuis la base de données.
- **Excel (Macros & TCD) :** Création du reporting et d'un tableau de bord interactif.

## 📂 Contenu du Dépôt
- `Livrables_SQL/` : Contient les requêtes SQL ayant permis d'extraire les données nécessaires.
- `Reporting_Excel/` : Contient le fichier de reporting final `.xlsm` (Tableaux Croisés Dynamiques et Macros).
